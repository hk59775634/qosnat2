## 概要

FQDN 防火墙别名支持从 URL 拉取域名列表，保存与后台刷新时自动解析为 IPv4 成员。

## 新增

- FQDN 别名可设置 `url`：HTTP(S) 拉取域名列表（每行一个），再由网关 DNS 解析写入 `members`
- Web 别名页：FQDN 类型增加 URL 输入；列表展示 URL 与拉取时间
