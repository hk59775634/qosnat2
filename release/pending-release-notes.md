## 概要

支持 FQDN 防火墙别名：网关主动解析域名并用于出站分流，无需客户端使用网关 DNS。

## 新增

- 防火墙别名支持 `type: fqdn`（`domains` 列表），网关经系统 DNS 解析 A 记录写入 `members`
- 后台约每 5 分钟自动刷新 URL / FQDN 动态别名；手动刷新 API 同步支持 FQDN
- Web「防火墙别名」页可创建与刷新 FQDN 别名

## 优化

- 动态别名成员变更时，先按旧 members 清理再重放出站策略 `ip rule`，避免遗留陈旧目的地址规则
