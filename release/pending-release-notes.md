## 概要

接口 IP 可经 netplan 持久化管理，新增 NAT/转发用虚拟 IP，WireGuard/OCServ 默认地址迁出 10.0.0.0/8

## 新增

- 网络 → 接口：可增删改物理网卡 IPv4（写入 state 与 `/etc/netplan/99-qosnat2.yaml` 并 apply）
- 网络 → 虚拟 IP：IP Alias，专供出站 NAT / 端口转发绑定；出站 NAT 与端口转发页可选用

## 优化

- OCServ 默认地址池改为 `198.18.250.0/24`（新建/空地址回填）
