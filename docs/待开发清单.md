## 已知限制

- 鉴权：Session Cookie + `X-API-Key`（无 RBAC / MFA）
- 防火墙：复杂规则勿与手写 `nftables-qosnat.nft` 混用后被 `Apply` 覆盖
- 多 WAN 同 metric 才合并 **nexthop weight**（不同 metric 仍按 metric 选路）
- 未建 profile 的源网段 **不会** ingress mirred（仅 `shaperMirredCIDRs` 列表内网段）
- VPN：WireGuard + ocserv（OpenConnect）；无 IPsec/OpenVPN 守护进程
- HTTPS 变更需 **root** 下自动 `systemctl restart qosnatd`，否则 UI 返回手动重启提示
