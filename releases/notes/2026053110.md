# qosnat2 2026053110

## 概要

Web Terminal 移至高级设置导航；修复常规保存误触发 TLS 重载导致 qosnatd 退出

## 新增

- （无）

## 优化

- Web Terminal 入口调整至 **系统 → 高级 → Web Terminal**

## 修复

- 常规设置保存时不再每次提交 `tls_enabled`，避免误判 TLS 变更并 reload 进程
- 后端仅在 TLS/ACME 实际变更时才 apply TLS

## 删除

- 常规设置中「启用 Web Terminal」复选框（改为导航直达）
