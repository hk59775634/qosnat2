# qosnat2 2026072008


## 概要

添加独立 IP 代理后自动连接测试，并像 WARP 一样展示出口 IP 与国家/城市信息

## 新增

- 经 TUN 探测代理出口 IP 与地理位置（ipwho.is / ip-api / Cloudflare trace）
- 添加 ProxyEgress 后若已安装 sing-box，自动发起连接与有效性测试
- 列表与任务结果展示 `exit_info`（IP · 城市/地区/国家、ISP、检测时间）

## 优化

- 连接/测试失败时停止隧道并返回明确错误，避免无效代理长期占用 TUN
