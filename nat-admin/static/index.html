全局默认限速（每 IP）

LAN 口所有 IPv4 流量均参与限速；未匹配网段/单 IP 规则时使用本默认，每 IP 独立令牌桶

网段批量限速

例：10.100.0.0/16 → 15mbit（任意 IPv4 网段；最长前缀优先于默认）
